Проект представляет собой реализацию контейнеров STL(vector, list, stack, string, unordered_map), 
алгоритмов(move, swap, hash для хэш-мапы), умные указатели(unique_ptr).
С помощью них реализовано решение таких задач как TwoSum, First Unique Character, Move Zeroes, Valid Parentheses,
что доказывает работоспособность моей реализации.

Сборка и запуск:
Проект не требует внешних зависимостей. 
Запустить в IDE можно через Cmake, файл зауска прикрепил к проекту.
Запуск через терминал: 
Компиляция: 
g++ main.cpp -o my_stl_app
Запуск: 
./my_stl_app

Особенности реализации:
- СТрогое соблюдение  Rule of Five (конструкторы копирования/перемещения, деструктор, операторы присваивания).
- Отсутствие утечек памяти (проверено логикой RAII).
- Использование шаблонов (templates) для универсальности контейнеров.
